<p>URL化。编写一种方法，将字符串中的空格全部替换为<code>%20</code>。假定该字符串尾部有足够的空间存放新增字符，并且知道字符串的“真实”长度。（注：用<code>Java</code>实现的话，请使用字符数组实现，以便直接在数组上操作。）</p>

<pre>
<strong>输入</strong>："Mr John Smith    ", 13

<pre>
<strong>输入</strong>："               ", 5

</pre>

<p><strong>提示：</strong></p>
